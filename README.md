# Pokedex Lite 🚀

A modern, high-performance, and premium Pokemon web application built with **Next.js 16** and **React 19**.

## 🚀 Deployment (Vercel)

The project is structured with the Next.js application inside the **`frontend`** directory.

### IMPORTANT for Vercel:
1.  **Root Directory**: In Vercel Project Settings, set the **Root Directory** to `frontend`.
2.  **Framework**: Next.js (Automatic).

## 🛠️ Tech Stack
- Next.js 15+ (App Router)
- React 19
- Vanilla CSS (Modules)
- Framer Motion
- PokéAPI

## 📦 Getting Started
1. `cd frontend`
2. `npm install`
3. `npm run dev`
